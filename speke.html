SPEKE protocol

SPEKE protocol uses given key and password to encode the message. Each of two people based on password and generated random bits gets their own code that is checked with each recieved message. In case the encoded message doesn't kontain the generated key the conversation has been interrupted, so the program returns an error telling that. The program is built in Python.
